# Spotify Songs Popularity Prediction using SVM

## Project Overview
This project predicts whether a Spotify song is popular using the Support Vector Machine (SVM) algorithm.

## Dataset
Spotify Tracks Dataset from Kaggle.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
- Support Vector Machine (SVM)

## Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Feature Scaling
6. Model Training
7. Prediction
8. Model Evaluation
